# Protocol Overview

::: tip Work in Progress
This page is under construction. Check back soon for the full protocol specification.
:::

The OHAI protocol defines a standardized way for AI agents to discover, query, and control smart home devices regardless of their manufacturer or underlying platform.

## Key Concepts

- **Messages** — The fundamental unit of communication
- **Devices** — Abstracted representations of physical hardware
- **Capabilities** — What a device can do (e.g., toggle, dim, set temperature)
- **AI Agents** — Intelligent controllers that interpret user intent
