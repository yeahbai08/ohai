**OHAI** (Open Home AI) is an open protocol that uses AI to control and access smart home devices. It aims to eliminate the ecosystem barriers created by IoT device manufacturers and platform companies.

## Goals

- **Vendor Neutrality** — Work with devices from any manufacturer
- **AI-Driven** — Use large language models for natural, flexible device control
- **User-Centric** — Let consumers control devices according to their own habits
- **Interoperability** — Enable intelligent cross-brand device automation

## Why OHAI?

Today's smart home landscape is fragmented. Each manufacturer builds its own walled garden, forcing consumers to choose a single ecosystem or juggle multiple apps. OHAI changes this by introducing a universal, AI-powered protocol layer that sits between the user and their devices.
